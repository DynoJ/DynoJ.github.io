About Me

I’m a software engineer with a Bachelor’s degree in Computer Science and a minor in Business Administration from Texas State University. My foundation lies in full-stack development, where I’ve built and deployed web applications using Python (Flask), Firebase, and modern front-end frameworks like React and Vite.

Throughout my academic journey and personal projects, I’ve tackled real-world problems across system administration, network management, and embedded computing using Raspberry Pi. These experiences taught me the value of clear documentation, security-conscious design, and user-first thinking.

I’ve also explored data analytics and cybersecurity, developing skills in SQL/MySQL, statistical modeling, and network traffic analysis. My work includes building data pipelines, automating tasks, and designing proactive security systems. I stay current with emerging tools and practices to ensure I can build resilient, future-ready solutions.

I’ve contributed to team-based hackathons and university organizations like Boko Hacks, IEEE, and the Google Developer Student Club — gaining exposure to collaborative development and modern software workflows. I'm committed to continuous learning, and I’m excited to bring my full-stack skill set to roles in software engineering, systems integration, or adjacent tech domains.
