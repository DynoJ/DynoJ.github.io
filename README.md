## 🚀 Quick Start

1. Clone the repository
```bash
git clone https://github.com/DynoJ/DynoJ.github.io.git
```

2. Open locally
```bash
# Using Python
python3 -m http.server 5500

# Or simply open index.html in your browser
```

3. Navigate to `http://localhost:5500`
